Galactic Gunner

Introduction

Galactic Gunner is a captivating 2D top-down shooter game that showcases my game design and programming skills. Developed entirely in C++ using a provided framework, this project was a standout entry in the 'Develop at Ubisoft' competition, earning me an interview opportunity.

By utilizing object-oriented programming principles, I encapsulated all data and behaviors related to each game object. This approach not only streamlined code management and modification but also facilitated code reuse through the implementation of generic base classes and inheritance.

I employed the singleton design pattern for managing game states to ensure efficient state control. This solution allows seamless access to the state manager from any code section, enabling easy switching between the states.

With a focus on minimizing code duplication, I implemented nine distinct enemy types, each deriving from a base enemy class. These enemies exhibit diverse behaviors while intelligently avoiding collisions with each other. The complex enemy design really enhances the gameplay immersion and experience.

To enhance gameplay variety and ensure a fun dynamic experience, I structured the game progression into waves, introducing a new wave every 30 seconds. The flexibility of this design choice enables effortless addition, removal, and modification of waves, making level design convenient and seamless.

Other Features

Upgrade System: Players can enhance their ship's performance by collecting gems dropped by defeated enemies. The upgrade system, coded for quick modification, ensures a rewarding progression curve and enables seamless adjustments to player abilities and weapon upgrades.

Save and Load: I integrated a simple yet useful save and load system, allowing players to track their highest and overall records. This feature enhances engagement and encourages players to beat their previous records.

Game Description

Traverse through galaxies on your spaceship and brace yourself for thrilling battles against various enemy ships.

Use your shooting skills and maneuver through waves of enemies while strategically collecting their loot to enhance your ship.

Survive relentless waves of enemies as the timer counts down to successfully beat the game.

Galactic Gunner - Solo Project Selected for Interview by 'Develop at Ubisoft'

My solo project for 'Develop at Ubisoft' over the weekend. It was selected by the judges, and I was invited for an interview. Galactic Gunner is a 2D top-down shooter game that I designed and developed entirely in C++ using the provided framework. It displays the theme of the contest which was space exploration.

I used object-oriented programming to encapsulate all data and behaviour related to each game object. This makes it easier to manage and modify code related to each game object. It also makes it possible to use inheritance and reuse code by creating generic base classes that others can derive from. I decided to use singleton design pattern to manage different states of my program. This works well for my game and provides easy access to the state manager from anywhere in the code. Through the one shared instance of the state manager, you can switch to different states of the game. There are 9 different enemy types with various behvaiours. They all derive from the base enemy class to minimize code duplication. The enemies avoid each other while engaging in their designated behaviour. I implemented the game progression in waves, so that there is a new wave every 30 seconds. Waves of the game can easily be added, removed, and modified. This makes level design so convenient. I added save and load feature to keep track of player's highest and overall records. Player can upgrade it's performance by picking up gems that enemies drop. I coded the upgrade system so that I can modify it in a matter of seconds.

Traverse through galaxies on your spaceship and get ready for battle. You will face different enemy spaceships that will try to take you down! Defeat them while dodging their attacks and collect their loot to enhance your ship and performance. Survive the enemy waves and you will win when timer hits zero. There is a new wave of unique enemies every 30 seconds. There are a total of 9 game waves that player needs to go through. The enemies spawned in each wave are relatively stronger than the previous ones. The spawn rate of enemies goes higher as the game goes on.

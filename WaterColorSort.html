Water Color Sort

Introduction

Water color sorting game re-created in Unity. Play the first 10 levels online!

Water Color Sort is a puzzle game where you move the colors inside of tubes around to have only the same color in each tube. Think logically and strategize and find your own way to sort the colorful water. I have 10 levels on this game and I implemented a re-do button in case you get stuck. It's okay, that happens!

You can select tubes by clicking on them. You can only pour into tubes that have space and that their top color matches the selected tube's top color. This game is super addictive and challenging at the same time!

This project was a great learning experience. I practiced and used a lot of shader graphs and animation curves to make the fill and pour effects happen. I also created the confetti visual effect from scratch which was fun.
